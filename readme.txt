= Documentation & Support =

Got a problem or need help with CProtal? Head over to our [documentation](http://docs.cportal.com/) and perform a search of the knowledge base. If you can’t find a solution to your issue then you can create a topic on the [support forum](https://wordpress.org/support/plugin/cportal/).

== Frequently Asked Questions ==

= Where can I find CPortal documentation and user guides? =

For help setting up and configuring CPortal please refer to our [documentation](http://docs.cportal.com/)

= Where can I get support or talk to other users? =

If you get stuck, you can ask for help in the [CPortal Support Forum](https://wordpress.org/support/plugin/cportal).

= Will CPortal work with my theme? =

Yes! CPortal will work with any theme, but may require some styling or making [custom templates](https://docs.cportal.com/article/1570-templates-structure).

= Will CPortal work with WordPress Multisite installation? =

Yes! CPortal is WordPress Multisite compatible.

== Installation ==

1. Activate the plugin
2. That's it. Go to CPortal > Settings to customize plugin options
3. For more details, please visit the official [Documentation](http://docs.cportal.com/) page.
