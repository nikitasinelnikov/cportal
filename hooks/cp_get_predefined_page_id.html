Filter: cp_get_predefined_page_id

apply_filters( 'cp_get_predefined_page_id', $page_id, $slug ) → {string}

Filters the predefined page ID.

Note: See all predefined pages slugs in CP()->config()->get( 'predefined_pages' );

Note: CPortal internally uses this hook for getting integrated with multilingual plugins that make duplicates of the pages for translations.

Parameters:
Name Type Description
$page_id string

The predefined page ID. The value obtained from options.
$slug string

The predefined page slug. E.g. 'my-portal'.

The predefined page ID.
